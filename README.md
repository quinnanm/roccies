# roccies
roc and histogram plotting stuff

How to use:


```
#first time
conda create --name roccies python=3.7
conda activate roccies

#python packages
pip install numpy pandas scikit-learn scipy matplotlib tqdm PyYAML

#uproot
pip install uproot awkward
pip install lz4 xxhash

#histogram stuff
pip install mplhep
pip install "hist[plot]"

#may not be needed but just in case
#conda install notebook
conda install jupyter

jupyter notebook

#after install 
conda activate roccies
jupyter notebook

```
